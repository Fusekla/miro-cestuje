# Design Direction

## Core Mood

The website should feel like a small travel magazine, not a planning tool or admin app.

Target feeling:

- modern
- sleek
- warm
- editorial
- calm
- premium without looking luxurious for its own sake

## What The Design Must Communicate

- this trip is thoughtfully planned
- the information is practical
- the experience is relaxed and worth browsing

## Visual Principles

### Photography First

- every trip should have a strong cover image
- each day should have a highlight image when available
- images should feel integrated into the layout, not bolted on

### Strong Typography

- typography should carry a lot of the visual identity
- headlines can feel expressive and editorial
- body text should remain easy to scan and friendly on mobile

### Spacious Layout

- prefer fewer, larger sections
- use whitespace to separate content
- avoid cramped cards and tiny utility labels everywhere

### Soft, Earthy Palette

The current direction is warm paper, deep forest, muted clay, and restrained accent tones.

This may evolve, but the palette should stay:

- grounded
- natural
- low-noise
- readable

## Interaction Principles

- collapsed day summaries first
- details only when requested
- subtle motion only
- transitions should support rhythm, not draw attention to themselves

## Mobile Rules

- mobile is first-class, not a fallback
- cards and sections should still feel premium on smaller screens
- no tiny text, awkward side-by-side layouts, or over-complicated controls

## What To Avoid

- dashboard UI
- generic startup landing page patterns
- bright default blues and purples
- cluttered badges everywhere
- giant blocks of unstructured text
- too many icons
- novelty animations
- feature creep disguised as polish

## Content Presentation Rules

- the collapsed day should answer the "what kind of day is this?" question
- the opened state should answer the "what exactly happens?" question
- accommodation and drive time should feel useful, not dominant

## Quality Bar

Before accepting a design change, ask:

1. Does it feel more like a travel editorial page or more like a tool?
2. Does it improve clarity without adding noise?
3. Does it still look intentional on mobile?
4. Would a friend actually enjoy opening this page?
